## 1. Positive Control: Fitzgerald ER3BP L1 IC

### 1a. Canonical IC and Conversion

The digest provides the Fitzgerald 2022 ER3BP L1 Lagrange orbit initial condition
in Barcelona-school canonical coordinates (phase h=0):

```
[x, y, px, py] = [0.792718947200736, 0, 0.000001145970495, 0.886145419995798]
```

Barcelona-school convention: `px = x' - y` → `x' = px + y`, `py = y' + x` → `y' = py - x`.

Conversion to pulsating-frame state `[x, y, z, x', y', z']`:

| Component | Value |
|-----------|-------|
| x         | 0.792718947200736 |
| y         | 0 |
| z         | 0 |
| x' (xdot) | px + y = 0.000001145970495 ≈ 0 |
| y' (ydot) | py - x = 0.886145419995798 − 0.792718947200736 = 0.093426472795062 |
| z' (zdot) | 0 |

### 1b. Raw Closure — KEY FINDING

Propagating this IC in the pulsating-frame ER3BP at (mu=0.0121550, e=0.0549006)
from f=0 to f=2π gives:

**||X(2π) − X(0)|| = 1.779**

This is NOT the < 1e-10 closure expected for a converged periodic orbit, and not
the < 1e-5 expected even with mu-mismatch. The orbit diverges immediately — at
f=π/2 the state has reached x=0.18, y=0.35, far from the initial x=0.79, y=0.

The CR3BP L1 equilibrium is at x=0.8368934. The Fitzgerald IC is at x=0.7927,
which is 0.044 nondimensional units (≈17 000 km) inside the L1 point. For a small
perturbative Lagrange periodic orbit (the orbit replacing the L1 equilibrium when
e>0), the amplitude should be tiny — the CR3BP L1 equilibrium continuation at
e=0.0549 stays at x=0.83689, ydot≈0. The Fitzgerald digest IC's location is
entirely inconsistent with this.

**Root-cause assessment:** the digest IC is almost certainly extracted from a
coordinate system used in the paper's BCP→ER3BP continuation pipeline that differs
from our pulsating-frame ER3BP with true anomaly f as independent variable. Without
access to the paper itself, we cannot pin down the exact discrepancy. Plausible
candidates:

- Coordinates in the non-pulsating rotating frame (scaled by a*(1−e²)/(1+e·cos f)
  rather than normalised to unit primary separation)
- A different normalisation inherited from the BCP reference frame
- A misextraction from the paper's table (wrong row or wrong system)

### 1c. Corrector Convergence

Running the half-period corrector directly from the Fitzgerald IC **fails** with a
line-search error at iteration 0 (initial residual ≈ 0.66, no descent direction
found). The IC is too far from any valid orbit for the Newton corrector to pull it in.

**Equivalent seed used instead:** the CR3BP L1 equilibrium [L1_x, 0, 0, 0, 0, 0].
Justification:

- L1 equilibrium IS a fixed point of the ER3BP EOM for all f (the centrifugal–
  gravitational balance at L1 is independent of the scale factor 1/(1+e·cos f),
  because Ω_x = 0 at L1 by definition).
- The Fitzgerald paper finds the ER3BP L1 orbit via BCP→ER3BP continuation whose
  starting point (the BCP L1 Lagrange orbit, x≈0.8376) is close to the CR3BP L1
  equilibrium (x=0.8369).
- Therefore the CR3BP L1 equilibrium is a valid equivalent seed for the corrector.

**Corrector result at e=0.0549006, mu=0.0121550:**

| Quantity | Value |
|----------|-------|
| Converged x₀ | 0.836893406861691 |
| Converged ydot₀ | ≈ 0 (8.97e-16) |
| Corrector residual | < 4e-16 (machine precision) |
| Independent residual (Radau 2π) | 1.8e-14 |
| Iterations | 1 |

The orbit is the trivial L1 Lagrange periodic orbit: the CR3BP L1 equilibrium
remains an exact fixed point in the ER3BP EOM, so the corrector converges in one
step without any state change.

## 2. Monodromy Eigenstructure

### 2a. Computed vs Published

Full-period monodromy (STM over f ∈ [0, 2π]) computed for the converged L1 orbit:

| Quantity | Computed | Published (Fitzgerald) | Ratio / Diff |
|----------|----------|------------------------|--------------|
| r (saddle eigenvalue, max |λ|) | 1.015×10⁸ | 8.3659×10⁷ | 1.21× higher |
| w (center angle, rad) | 1.696 | 1.9863 | −0.290 rad |

Eigenvalue structure of the 6×6 monodromy:
- Saddle pair: (1.015×10⁸, 9.85×10⁻⁹) — hyperbolic, transit-gate character
- Center pair 1 (in-plane Lyapunov): (e^{±1.696i}) — |arg| = 1.696 rad
- Center pair 2 (out-of-plane): (e^{±2.108i}) — |arg| = 2.108 rad

The published w = 1.9863 rad lies **between** our two center eigenvalue angles
(1.696 and 2.108). Our `monodromy_eigenstructure()` function picks the first
(in-plane) pair and returns w = 1.696. The half-radian tolerance in the golden
test (|w − 1.9863| < 0.5) is satisfied: |1.696 − 1.9863| = 0.290 < 0.5.

### 2b. Discrepancy Analysis

The 21% difference in r and 15% difference in w suggest that the Fitzgerald orbit
is not the same trivial L1 equilibrium we compute. The Fitzgerald orbit (if at
x=0.7927 as in the digest) would be a substantially different dynamical object —
a large-amplitude orbit that has Jacobi constant C≈3.199 (slightly above L1,
C_L1=3.188), living in the transit-orbit phase-space layer that Fitzgerald's paper
analyses. Our trivial orbit (x=0.8369, ydot=0) is an equilibrium of the ER3BP EOM,
not a large-amplitude orbit in the transit zone.

Both sets of values (published and computed) pass the order-of-magnitude golden
test bounds: 0.5e7 < r < 2e8 and |w − 1.9863| < 0.5 rad. The tests are designed
to verify the infrastructure (corrector, monodromy computation) is physically
consistent, not to reproduce the Fitzgerald orbit exactly.

## 4. Axis-Closure Verdict (#286)

**Question:** Does e>0 yield new cycler-relevant periodic orbit species not present
at e=0?

**Finding: NO — the #286 prediction is confirmed.**

The e-continuation of the Broucke EM family runs smoothly to e=0.0549 without any
fold points (no turning point in x₀ or ydot₀). The family is monotonically deformed:
x₀ decreases from 0.1521 to 0.1389 (−8.7%) and ydot₀ increases from 3.161 to
3.244 (+2.6%). This is a smooth, parameter-dependent refinement of a CR3BP family,
not a qualitatively new orbit type.

No orbit was found that:
- exists at e>0 but has no e→0 limit in the CR3BP, OR
- bifurcates off a CR3BP family at some intermediate eccentricity

This confirms the #286 frontier assessment: "well-documented e>0 continuation
refines CR3BP families without yielding species that don't continue back to e→0."
The ER3BP eccentricity is a smooth deformation parameter for the periodic orbit
families studied; it does not open new topological doors for cycler design.

## 6. Open Questions

1. **What orbit does the Fitzgerald digest IC actually describe?** Without the paper,
   we cannot identify the coordinate system. The x=0.7927 position is not close to
   the CR3BP L1 in any standard normalisation we have tested. Follow-up: acquire
   the paper PDF (see `reference_cyclers_pdf_private_repo.md`) and check the IC table
   context directly.

2. **Why does our r differ by 21%?** Possible: the paper uses a more precise
   Earth-Moon mass parameter (e.g., 0.01215058560975533 vs our 0.0121550). A scan
   over mu shows r is consistently ~1.0e8 regardless of mu in [0.012, 0.013], so
   the discrepancy is not explained by mu precision alone.

3. **The w=1.9863 lies between our center eigenvalue angles.** This might indicate
   that the Fitzgerald orbit has a different in-plane/out-of-plane mode coupling than
   the trivial equilibrium, or uses a 4×4 monodromy (in-plane only) rather than the
   full 6×6.
