# Digest: Touring the Saturnian System (Wolf 1996)

**Paper**: `papers/wolf-1996-touring-saturnian-system-spie.pdf`
**Authors**: Aron A. Wolf (Jet Propulsion Laboratory)
**Venue**: SPIE Vol. 2803 (1996)
**URL/DOI**: 10.1117/12.254089 (approx, via SPIE)

## Verdict

**HONEST NEGATIVE**. The paper provides a high-level description of the Cassini tour design objectives (Saturn, Magnetosphere, Titan, Icy Satellites, Rings) and presents a 4-year sample tour. While it contains a full schedule of flybys (Table 2), it **does not** publish the per-flyby $V_\infty$ (hyperbolic excess velocity) data or detailed state vectors required for catalogue entry. 

## Key Contents

1. **Tour Design Constraints**: Discusses the trade-offs between different scientific objectives, such as lowering the periapsis for ring observations vs. raising it for magnetotail coverage.
2. **Sample Tour (Table 2)**: Provides a detailed sequence of 48 targeted flybys (42 Titan, 6 icy moons). For each flyby, it lists:
   - Encounter body
   - Date and time (e.g., `20041127.141104`)
   - Altitude (km)
   - Latitude (deg)
   - Post-flyby period (days)
   - Post-flyby inclination to Saturn equator (deg)

## Notes for the Cyclers Project

- This paper is a useful summary of the *geometry* and *sequencing* of the Cassini tour, but lacks the astrodynamic energy metrics ($V_\infty$, $\Delta V$) needed for our trajectory models.
- Since we bypassed the need for a 1990s Cassini paper by querying NASA JPL Horizons directly to calculate the actual flown $V_\infty$ values (`2026-06-19-digest-cassini-vinf.md`), this paper serves as contextual background rather than a primary data source.
