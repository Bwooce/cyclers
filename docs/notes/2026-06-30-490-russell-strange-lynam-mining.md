## Lynam-Longuski 2011 documented IEG triple cyclers (the #480 cross-check)
- **Indefinitely-repeatable "GIPEIPE" cycler:** Ganymede-Io-(perijove)-Europa-Io-(perijove)-Europa,
  period 3.5 d / repeats every 7.05 d (the Laplace period); MALTO-optimised low total ΔV; the
  lowest-ΔV member.
- **Single-period IEG triple cycler:** Europa→(perijove)→Io→Ganymede→(apojove)→Europa, 7.05 d,
  example epoch depart 2024-08-27, **total ΔV ≈ 11 m/s** (patched-conic ephemeris); ballistic
  version needs a sub-surface (−175 km) Europa flyby so it's the powered member.
- **#480 cross-check (the key result):** these are **DISTINCT sequences from EGGIE (E-G-G-I-E) and
  EIGE (E-I-G-E)** — GIPEIPE and E-I-G-E... actually L-L's single-period E-(P)-I-G-(A)-E is the SAME
  3-flyby topology as #480's EIGE but with explicit perijove/apojove nodes and an 11 m/s powered
  Europa flyby (vs our ideal ballistic EIGE). So **Lynam-Longuski is a 2nd, independent, EARLIER
  (2011) source for the IEG triple-cycler class** (Hernandez 2017 cites it as ref [7]); its 11 m/s
  single-period IEG ΔV is an independent low-ΔV anchor consistent with #480's findings. It does NOT
  duplicate EGGIE; it CORROBORATES the IEG class + adds the GIPEIPE member.
