# docs/refs — reference paper pointers

Source PDFs (AIAA / JSR / NASA-JPL / dissertations) are **not stored in this
repository**: they are copyrighted/paywalled and must not be redistributed.

Every paper used as a reference or golden-anchor source is cited in
[`data/catalogue.yaml`](../../data/catalogue.yaml) by **full title + DOI** (see
each entry's `first_published` / `corroborating_sources`), and in
[`data/OUTSTANDING.md`](../../data/OUTSTANDING.md) for sources still being sought.

To work with a paper, obtain it from the publisher / NASA NTRS / your library
using the DOI in the catalogue entry.
